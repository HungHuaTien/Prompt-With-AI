# 🧠 網站資源總覽

網址: https://github.com/HungHuaTien/Prompt-With-AI/

歡迎來到AI Coach教你學習的實驗站，這裡是Prompt With AI，專門提供提示語教案。  
所有內容依照用途與功能分類為不同子資料夾，方便未來快速定位、版本控制與共享。

> ✨ **原則：Less is more, every file has a home.**  
> 無論是學習筆記、心智圖還是文章，都能找到對應的分類。

---

## 📁 資料夾結構與說明

### 📝 `_posts/` 貼文與推廣文

- 本站用的發文原稿
- 含貼文標題、日期與用途說明

### 🧭`assets` 貼文資源

- ✅ `/images/` — 貼文圖檔


### 📘`courses`課程教案

- ✅`/AIL/` — AI素養，AI Literacy 
- ✅`/Python/` — 程式語言
- ✅`/R/` — 資料視覺化，敘述分析
- ✅`/Modelling/` — 建模分析，因果分析與預測分析



### 📦 `_files` — 可供輸出下載的檔案

- 定稿、已排版或轉換格式版本
- ✅`/handouts/` — 給學生
- ✅`/notes/` — 給老師
- ✅`/figures/` — 圖檔



---

## 🔧 使用建議

- 所有檔案建議加上 YAML metadata，便於後續分類與查找
- 每個資料夾皆可搭配 `README.md` 作為說明與索引
- 若有需要，可延伸整理為 Quarto 網頁或 GitHub Pages 教案網站
